# Política de segurança

## Versões suportadas

Apenas o ramo `main` deste repositório é considerado para correções de segurança ativas.

## Reportar uma vulnerabilidade

Se encontrar um problema de segurança relacionado com este tema (por exemplo, XSS em templates Liquid, exposição de dados em snippets ou configuração insegura), **não** abra um issue público com detalhes exploráveis.

1. Abra um [**Security Advisory**](https://github.com/19820707/temusyy/security/advisories/new) no GitHub (recomendado), ou  
2. Contacte o mantenedor do repositório por um canal privado, com descrição do impacto e passos mínimos para reprodução.

O objetivo é corrigir ou mitigar antes de divulgação pública coordenada.
